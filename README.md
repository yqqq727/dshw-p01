### 数据来源
- **股票行情**：Baostock，后复权，日度
- **市场指数**：
  - 沪深300（000300）：作为CAPM分析的市场基准
  - 中证500（000905）：代表中小盘股票，选择理由是中小盘股票波动性较大，可对比沪深300分析不同市值风格差异
- **宏观指标**：
  - **CPI同比增速**（必选）：反映通胀水平，影响货币政策预期，来源：Akshare (ak.macro_china_cpi_yearly)
  - **人民币/美元汇率**（自选）：汇率变化影响出口企业盈利和外资流向，与股市相关性较强，来源：Akshare (currency_boc_sina)
- **财务数据**：最近5个年度（2019-2023）的ROE、净利润率、资产负债率，数据来源：Baostock

### 存储方式
- **基础**：CSV（方式 A）
  - **优点**：通用性强，几乎所有工具都支持；人类可读，便于检查和调试；便于版本控制和diff比较
  - **在什么情况下会力不从心**：文件体积大，压缩效率低；读取速度较慢，尤其大文件；不保留数据类型信息，需重复转换；不支持列式读取，必须加载整个文件
- **进阶**：Parquet（方式 B）
  - **选择理由**：列式存储只读取需要的列内存效率高；压缩率高相比CSV节省约50%存储空间；读取速度快比CSV快2-3倍尤其大文件优势明显；保留Schema数据类型信息持久化避免重复转换；生态兼容与Pandas/Spark/PyArrow无缝集成
  - **对比说明**：在本次数据规模（15,160行）下，CSV读取耗时0.016秒（1,127.5 KB），Parquet读取耗时0.006秒（542.7 KB），差异已经显现；当数据规模扩大到百万级或千万级时，Parquet的优势会更加显著

## 项目说明

### 数据规模
- **股票数据**：10只股票 × 1516个交易日 = 15,160条记录
- **指数数据**：2个指数 × 1516个交易日
- **宏观数据**：CPI (477条记录)、美元汇率 (180条记录)
- **财务数据**：10只股票 × 5年 × 3个指标 = 150条记录

### 数据清洗步骤
1. **缺失值检测**：统计每列缺失值的数量和比例，分析缺失原因
2. **缺失值处理**：向前填充（ffill），适合连续交易日数据，保证时间序列连续性
3. **日期格式统一**：转换为datetime类型并设为索引，便于时间序列分析
4. **数据类型检查**：数值列转换为float/int类型，确保可进行数学运算
5. **重复值处理**：检测并删除重复行，保证数据唯一性
6. **离群值标注**：计算日收益率，标注±20%极端值，用于风险分析

### 分析发现
- **收益分化明显**：比亚迪（12.35%）、中国石油（11.87%）、贵州茅台（4.32%）表现较好；房地产板块跌幅最大（万科A -35.09%）
- **CAPM分析**：比亚迪具有显著正Alpha（α=0.0015, p=0.009）；五粮液和比亚迪属于高β股票（β>1），符合周期性行业特征；中国石油R²最低（0.08），显示收益独立于市场

### 技术栈
- **数据源**：Baostock, Akshare
- **数据处理**：Pandas, NumPy
- **统计分析**：SciPy, Statsmodels
- **可视化**：Matplotlib, Seaborn
- **存储格式**：CSV, Parquet
