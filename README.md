## 二、数据清洗与管理（5分）

### 2.1 数据清洗流程（6步）

#### 步骤1：缺失值检测
- 统计每列缺失值的数量和比例
- 分析缺失原因（停牌、数据源问题等）
- 输出缺失值报告

#### 步骤2：缺失值处理
- **处理方法**：向前填充（ffill）
- **选择理由**：适合连续交易日数据，保证时间序列连续性，避免引入未来信息
- **验证**：处理后再次检测，确认无缺失值

#### 步骤3：日期格式统一
- 转换为 `datetime` 类型
- 设为索引（index），便于时间序列分析
- 确保日期升序排列

#### 步骤4：数据类型检查
- 数值列转换为 `float` 或 `int` 类型
- 确保可进行数学运算
- 移除千分位分隔符等格式字符

#### 步骤5：重复值处理
- 检测重复行（基于日期）
- 删除重复记录，保证数据唯一性
- 记录重复数量用于数据质量评估

#### 步骤6：离群值标注
- 计算日收益率：`r_t = (P_t - P_{t-1}) / P_{t-1}`
- 标注极端值：±20% 涨跌幅
- 用于风险分析和异常交易识别

## 三、存储方式说明（5分）

### 3.1 基础存储：CSV（方式A）

#### 优点
- **通用性强**：几乎所有工具都支持（Excel、Python、R、SQL）
- **人类可读**：便于检查和调试
- **便于版本控制**：文本格式，diff比较清晰

#### 局限性
- **文件体积大**：压缩效率低
- **读取速度慢**：必须解析整个文件
- **不保留类型**：每次读取需重新转换数据类型
- **不支持列式读取**：必须加载整个文件到内存

### 3.2 进阶存储：Parquet（方式B）

#### 选择理由
1. **列式存储**：只读取需要的列，内存效率高
2. **压缩率高**：相比CSV节省约50%存储空间
3. **读取速度快**：比CSV快2-3倍，尤其大文件优势明显
4. **保留Schema**：数据类型信息持久化，避免重复转换
5. **生态兼容**：与Pandas/Spark/PyArrow无缝集成

#### 性能对比（15,160条记录）

| 格式 | 文件大小 | 读取时间 | 写入时间 |
|------|---------|---------|---------|
| CSV | 1,127.5 KB | 0.016秒 | 0.045秒 |
| Parquet | 542.7 KB | 0.006秒 | 0.028秒 |
| **优势** | **-52%** | **-62%** | **-38%** |

### 3.3 存储策略

本项目采用**双格式存储**：
- **CSV格式**：用于数据交换、人工检查、版本控制
- **Parquet格式**：用于高效读取、大规模分析

当数据规模扩大到百万级或千万级时，Parquet的优势会更加显著。

### 4.2 CAPM模型分析

#### 模型设定
$$R_{i,t} - R_{f,t} = \alpha_i + \beta_i (R_{m,t} - R_{f,t}) + \epsilon_{i,t}$$

其中：
- $R_{i,t}$：股票i的日收益率
- $R_{f,t}$：无风险利率（使用3个月期Shibor代理）
- $R_{m,t}$：市场收益率（沪深300指数收益率）
- $\alpha_i$：超额收益（Jensen's Alpha）
- $\beta_i$：系统风险度量
- $\epsilon_{i,t}$：残差

#### 4.2.1 回归结果

| 股票 | Alpha (p值) | Beta | R² | Treynor比 |
|------|------------|------|----|----|
| 比亚迪 | **0.0015 (0.009)** | 1.28 | 0.42 | 0.062 |
| 五粮液 | 0.0003 (0.445) | **1.15** | 0.38 | 0.005 |
| 贵州茅台 | **0.0008 (0.012)** | 0.87 | 0.35 | 0.049 |
| 万科A | -0.0012 (0.087) | 0.92 | 0.28 | -0.013 |
| 招商银行 | 0.0004 (0.223) | 0.76 | 0.41 | 0.029 |
| 中国石油 | 0.0006 (0.156) | 0.52 | **0.08** | 0.064 |

#### 4.2.2 关键发现

1. **显著正Alpha**：比亚迪（α=0.0015, p=0.009）和贵州茅台（α=0.0008, p=0.012）存在显著超额收益
2. **高Beta股票**：五粮液（β=1.15）和比亚迪（β=1.28）属于进攻型股票，符合周期性行业特征
3. **低相关性**：中国石油R²最低（0.08），收益主要受油价驱动，独立于市场
4. **防御型股票**：招商银行（β=0.76）、中国石油（β=0.52）系统风险较低
