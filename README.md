## 项目概述

本项目对中国A股市场的10只代表性股票进行全面的量化分析，涵盖数据获取、清洗处理、描述性统计与可视化分析等完整流程。研究周期为2020年1月1日至2026年4月，旨在通过多维度分析揭示不同行业股票的收益风险特征、相关性结构以及与宏观因素的关系。

## 数据来源

### 股票行情
- **数据源**：Baostock
- **数据类型**：后复权日度行情
- **时间范围**：2020-01-01 至今
- **选择理由**：免费、稳定、无需token，数据质量高，覆盖A股全面

### 市场指数
- **沪深300 (000300)**：作为CAPM模型的市场基准，代表A股大盘蓝筹
- **中证500 (000905)**：代表A股中盘股，选择理由是中小盘股票波动性较大，可对比沪深300分析不同市值风格差异

### 宏观指标
- **CPI同比增速**：必选指标，反映通胀水平，影响货币政策预期
- **人民币/美元汇率**：选择理由是汇率变化影响出口企业盈利和外资流向，与股市相关性较强
- **数据来源**：Akshare (macro_china_cpi_yearly, currency_boc_sina)

### 财务数据
- **数据源**：Baostock
- **指标类型**：ROE（净资产收益率）、净利润率、资产负债率
- **时间范围**：最近5个年度（2019-2023）

## 存储方式

### 方式 A：CSV（基础格式）
**优点**：
- 通用性强，几乎所有工具都支持
- 人类可读，便于检查和调试
- 便于版本控制和diff比较

**缺点与局限性**：
- 文件体积大，压缩效率低
- 读取速度较慢，尤其大文件
- 不保留数据类型信息，需重复转换
- 不支持列式读取，必须加载整个文件

**适用场景**：中小规模数据（<100MB）、需要人工检查、多工具协作

### 方式 B：Parquet（进阶格式）
**选择理由**：
- **列式存储**：只读取需要的列，内存效率高
- **压缩率高**：相比CSV节省约50%存储空间
- **读取速度快**：比CSV快2-3倍，尤其大文件优势明显
- **保留Schema**：数据类型信息持久化，避免重复转换
- **生态兼容**：与Pandas/Spark/PyArrow无缝集成

**本次项目对比**：
- CSV：1,127.5 KB，读取耗时0.016秒
- Parquet：542.7 KB，读取耗时0.006秒
- 压缩比：约52%，读取速度提升约2.7倍

**差异显著性分析**：
在本次数据规模（15,160行）下，差异已经显现；当数据规模扩大到百万级或千万级时，Parquet的优势会更加显著，尤其在以下场景：
- 大数据集的特定列分析
- 频繁的读写操作
- 内存受限环境
- 需要快速原型开发

## 数据清洗步骤

1. **缺失值检测**：统计每列缺失值的数量和比例，分析缺失原因（停牌、节假日、数据源问题）
2. **缺失值处理**：向前填充（ffill），适合连续交易日数据，保证时间序列连续性
3. **日期格式统一**：转换为datetime类型并设为索引，便于时间序列分析
4. **数据类型检查**：数值列转换为float/int类型，确保可进行数学运算
5. **重复值处理**：检测并删除重复行，保证数据唯一性
6. **离群值标注**：计算日收益率，标注±20%极端值，用于风险分析

### CAPM 模型分析

**系统性风险（β）分层**：
- **低风险防御组**：中国石油（β=0.45）、兴业银行（β=0.70）
- **中性风险组**：招商银行（β=0.90）、贵州茅台（β=0.97）
- **高风险进攻组**：五粮液（β=1.29）、比亚迪（β=1.28）

**超额收益（α）分析**：
- **显著正α**：比亚迪（α=0.0015, p=0.009）- 具备真实超额收益能力
- **显著负α**：万科A（α=-0.0012, p=0.022）- 持续跑输市场
- **不显著α**：其余8只股票 - 收益主要由市场β决定

**拟合优度（R²）分析**：
- **最高R²**：五粮液（R²=0.49）- 收益高度依赖市场
- **最低R²**：中国石油（R²=0.08）- 收益独立于市场，防御属性明显

### 投资启示

1. **资产配置策略**：
   - 核心底仓：招商银行、兴业银行、中国石油（低β，稳定）
   - 进攻配置：比亚迪、五粮液（高β，高弹性）
   - 规避：房地产板块（持续下行，负α）

2. **风险管理**：
   - 行业分散：利用板块间低相关性降低组合波动
   - Beta管理：根据市场周期调整高β/低β资产配置比例
   - 极端风险：关注高β股票的回撤风险
